SUBJECT: API Integration Issue - Empty Pricelists & No Availability (JustRent)

Dear Renteon Support,

We are integrating the JustRent webshop via API and facing blocking issues with price calculation:

1. **Empty Pricelists**:
   `GET /api/pricelists` returns an empty array `[]`, despite having active pricelists (e.g., ID 351, 306).

2. **No Availability Error**:
   `POST /api/bookings/calculate` returns `422 Unprocessable Entity` with message:
   `"There are no available cars for given criterion."`
   
   We tested with:
   - Dates: Tomorrow, +30 days, +180 days (various durations).
   - Offices: IDs 53 and 54.
   - Pricelists: IDs 351 and 306.
   - Modes: `BookAsCommissioner: true` and `false`.
   - BookingSource: 114, 113.

Please verify if our API user has permissions to access these pricelists and perform calculations.
